# Elevation

We do have Primary, Secondary and Modal of elevation.

Primary show a shadow left,right,top and bottom
Secondary show a shadow only bottom such as Navbar

Each component that has an elevation effect (box-shadow) should be configurable with an elevation property.

Default = 0 elevation
